Sistema de Inventario Web
Descripción

Sistema de Inventario WEB desarrollado con Laravel y Bootstrap, diseñado para gestionar de manera eficiente productos, stock y movimientos dentro de un negocio. La plataforma permite registrar, actualizar y consultar inventarios en tiempo real, ofreciendo una interfaz intuitiva y responsiva gracias a Bootstrap.

Recreo Explore IQT
Descripción

El propósito de la app es ofrecer información sobre recreos turísticos en la ciudad de Iquitos. Está diseñada para guiar a los visitantes y/o gente local en la búsqueda de recreos turísticos, ofreciéndole información detallada sobre el sitio. Lo que se busca es que contenga información actualizada y detallada de los recreos turísticos en nuestra ciudad . Para el desarrollo de la aplicación se utilizó Flutter como framework y Dart como lenguaje de programación y para el manejo de datos se optó por implementarlo con SQLite. Este proyecto tuvo una duración de 50 días (8 semanas) y actualmente aún sigue en desarrollo ya que hay algunas partes por terminar.

Cajueziño App
Descripción

El proyecto abordado pretende mejorar el proceso interno de la empresa Cajueziño con respecto a los pedidos de sus productos, con la finalidad de mejorar la experiencia de los comensales en el lugar. Este proyecto fue desarrollado en un curso de la Universidad llamado Investigación, desarrollo e innovación y para su desarrollo lo que se hizo uso fue utilizar el lenguaje Dart y su framework Flutter, también del diseño de una base de datos relacional alojando los datos en el sistema de gesto de base de datos Microsoft SQL Server Management Studio y para poder hacer la conexión con la base de datos y manipulación de los datos se implementó Api's realizados con PHP donde las consultas eran realizadas en este lenguaje de programación.
